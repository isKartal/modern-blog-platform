# 🚀 Modern Blog Platform

A full-stack blog platform built with Django REST Framework and React.

## ✨ Features

- 🔐 JWT Authentication
- 📝 CRUD Operations for Posts
- 🔍 Search & Filter Posts
- 📱 Responsive Design
- 👤 User Management
- 📊 Admin Dashboard
- 🎯 Category Management
- 💬 Comment System

## 🛠️ Tech Stack

**Backend:**
- Django REST Framework
- SQLite Database
- JWT Authentication
- Django CORS Headers

**Frontend:**
- React.js
- Axios for API calls
- Modern CSS3
- Responsive Design
